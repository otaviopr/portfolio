# New portfolio site

This is the new GitHub Pages-ready portfolio, kept separate from the legacy Readymag export during the rebuild.

## Local preview

From this folder run `python3 -m http.server 8000`, then visit `http://localhost:8000`.

## Content

All public content lives in `content/site.json`. Each project has translations, sections, optional image paths and an optional `vimeo` ID. This is the file that the final admin will publish to GitHub.
